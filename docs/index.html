About the Project

The Shenlong Project simplifies the management of Kubernetes jobs through a robust API and CLI.

Key Features

  • Create Jobs
    Define and run Kubernetes jobs effortlessly.
  • Schedule Cron Jobs
    Automate recurring tasks with cron scheduling.
  • Monitor Jobs
    Retrieve job details in real-time.
  • Delete Jobs
    Clean up jobs with a single command.
  • Run as a Server
    Deploy as a standalone API server.

Usage Examples

shenlong create-job --name test --image busybox --command echo,hello
shenlong create-cron --name test --image busybox --command echo,hello
